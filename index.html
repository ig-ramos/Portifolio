Esta página web segue as principais regras de semântica do HTML5(para mais informações clique aqui), lembrando que, ainda estou no processo de aprendizagem da mesma, por tanto alguns erros podem vir a aparecer em algum momento no decorrer do código.

Para criar esta página utilizei das seguintes ferramentas. A primeira foi o editor de código-fonte, Visaul Studio Code desenvolvido pela Microsoft. A escolha desta ferramenta se deu por alguns fatores, tais como, a grande maioria dos produtores de conteudo que ensinam sobre Desenvolvimento Web Front-End, utiliza o Visual Studio Code. Além de que, esta ferramente é gratuita e de facíl aprendizado, o que possibilita um avanço mais rápido no aprendizado das tecnologias necessárias pra ser um Dev Web Front-End.

Além dele utilizei a ferramenta GIMP, um programa de código aberto que possibilita a edição e criação de imagens. Assim como o Visual Studio Code, o GIMP também é gratuito e de facil aprendizagem, o que para alguem que esta começando é algo muito importante.
